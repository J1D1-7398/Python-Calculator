# Python-Calculator
A multi calculator in Python using Tkinter library. I will try and solve any issues you have. I will update this repo regularly adding new calculators (I already have some in mind but if you want a specific one i will try and code it). Feel free to make any suggestions!

This program was only tested on Windows so it may not work on other systems. (Might work on Linux, I coded it so it does, but it might have some errors.)

## How to use the app:

Open Command Prompt and navigate to the directory where you have cloned/downloaded the repository. Remember to enter in the "Code" and the "APP" folder, where the python files are stored

When you have reached the directory in Command Prompt type
```
python main.py
```

✅ It should work fine!

The menu should pop up
